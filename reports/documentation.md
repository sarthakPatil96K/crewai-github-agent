## Overview
The Voice Assistant is a Python-based project that allows users to interact with their device using voice commands. It uses NLP techniques to understand and respond to user queries.

## Features
- Supports basic voice commands (e.g., "Hello", "What's the time?")
- Can play music and videos
- Can set reminders and alarms
- Can provide weather updates

## Installation
### Prerequisites
- Python 3.8 or higher
- pip
- git

### Installation Steps
1. Clone the repository using `git clone https://github.com/sarthakPatil96K/Voice_Assistant.git`
2. Navigate to the project directory using `cd Voice_Assistant`
3. Install the required packages using `pip install -r requirements.txt`
4. Run the project using `python main.py`
